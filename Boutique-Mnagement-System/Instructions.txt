-This project is about Boutique Management System in which all categories of users are handled.
-To compile this project, include all files in main and run the project.
-A welcome screen will appear.
-Click on the user type (ADMIN OR CUSTOMER).
-Enter a valid login ID and Password to login to your account.
-Now you can access all the features of this Management System.
